## 🚀 Featured Engineering Projects

### 1. [NYC Taxi Data ETL Pipeline & Orchestration](https://github.com/Shiva-Sharan/NYC-Taxi-Data-ETL-Pipeline-Data-Ingestion-Quality)
> *A production-grade, end-to-end Data Engineering pipeline for automated ingestion, transformation, and storage of massive NYC Taxi datasets.*

* **Business Impact:** Automated the processing of millions of taxi trip records, ensuring high data availability and quality for downstream analytics.
* **Architecture & Features:** Implemented modular **dbt** transformations on top of a **DuckDB** analytical engine. The entire workflow—from data extraction to testing—is orchestrated using **Kestra** and containerized via **Docker**.
* **Tech Stack:** Python, PostgreSQL, DuckDB, dbt, Docker, Kestra.
* **Engineering Concepts:** ELT Pipelines, Data Orchestration, Automated Data Quality Testing, Containerization.

### 2. [Enterprise Data Warehouse Implementation](https://github.com/Shiva-Sharan/DataWareHouse_Project_with_SQL)
> *A highly optimized SQL-based Data Warehouse showcasing dimensional modeling and complex analytical workflows.*

* **Business Impact:** Established a robust single source of truth for simulated enterprise datasets, drastically reducing query times for BI reporting.
* **Architecture & Features:** Designed a complete **Star Schema**, handling complex fact and dimension tables. Engineered advanced SQL queries (window functions, CTEs) to derive rapid business intelligence metrics.
* **Tech Stack:** SQL, Advanced Relational Databases.
* **Engineering Concepts:** Dimensional Modeling, Star Schema, Advanced SQL Analytics, Slowly Changing Dimensions (SCD).

### 3. [Real-Time AIoT Analytics & Facial Recognition System](https://github.com/Shiva-Sharan/Smart_Attendance_System_Using_Facial_Recognition_with_ESP32-CAM)
> *An edge-to-database analytics pipeline leveraging ESP32-CAM and computer vision for automated attendance tracking.*

* **Business Impact:** Replaced manual entry systems with a zero-touch, automated data logging infrastructure.
* **Architecture & Features:** Streams real-time visual data from an ESP32-CAM edge device. Uses **Python & OpenCV** for localized inference and logs structured timestamp records into an **SQLite** database for seamless dashboard integration.
* **Tech Stack:** Python, OpenCV, SQLite, ESP32-CAM, Edge Hardware.
* **Engineering Concepts:** Real-time Data Ingestion, Hardware/Software Integration, Computer Vision, Edge Analytics.

### 4. [RetailPulse Analytics Pipeline](https://github.com/Shiva-Sharan/Exploratory_Data_Analytics_with_SQL)
> *Comprehensive EDA workflows and SQL analytical queries to derive actionable insights from raw retail datasets.*

* **Business Impact:** Uncovered hidden sales trends and product performance metrics through rigorous data cleansing and exploratory analysis.
* **Architecture & Features:** Built heavily optimized relational database queries to clean dirty data, aggregate metrics, and generate ready-to-visualize statistical summaries.
* **Tech Stack:** SQL, EDA methodologies, Reporting.
